소나기 타자 연습기
2018115425 권오민

제가 만든 프로젝트는 소나기 타자 연습기입니다.

1. 프로그램을 시작합니다.
(cc pro.c -lcurses -lpthread -o pro)

2. game start, text file info, exit 중 원하는 메뉴를 선택합니다.
  2-1 game start
    easy mode, normal mode, hard mode 중 원하는 모드를 선택합니다.
    이때 다시 이전 페이지로 가고 싶으면 'b', 종료하고 싶으면 'q'를 누릅니다.
    모드를 선택하고 게임을 시작합니다.
    랜덤하게 10개의 단어를 줍니다.
    목숨은 총 3개가 주어지고, 게임이 끝나면 score를 알려줍니다.
    게임 도중 종료하고 싶으면 'quit'를 입력하면 됩니다.
    (이를 위해서 quit는 단어로 주어지지 않습니다.)
    'quit'를 입력하면 다시 mode 선택 화면으로 넘어갑니다.
  2-2 text file info
    easy.txt, normal.txt, hard.txt 중 보고 싶은 파일을 선택합니다.
    이때 다시 이전 페이지로 가고 싶으면 'b', 종료하고 싶으면 'q'를 누릅니다.
    파일을 선택하면 파일 안의 모든 단어를 보여줍니다.
    파일을 다 보았으면 enter 혹은 아무 문자나 입력하면 다시 list를 보여줍니다.
  2-3 exit
    3번을 선택하게 되면 게임이 종료됩니다.
    3초 뒤에 꺼진다는 알림을 보여주고 프로그램을 종료시킵니다.

3. 게임을 즐긴다.
